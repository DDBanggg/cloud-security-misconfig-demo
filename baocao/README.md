# Bao cao LaTeX

Thu muc nay chua source bao cao bang LaTeX.

## Cach bien dich

Nen bien dich bang XeLaTeX vi bao cao dung tieng Viet va font Times New Roman.

```bash
cd baocao
xelatex main.tex
bibtex main
xelatex main.tex
xelatex main.tex
```

Neu may chua cai LaTeX, co the dung TeX Live, MiKTeX hoac Overleaf.

## Cau truc

```text
baocao/
  main.tex
  references.bib
  chapters/
```
